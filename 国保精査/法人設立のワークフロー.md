# マイクロ法人設立・実務運用開始ガイド（ワーストケース・オンライン完結版）

## 1. 概要
本資料は、客先常駐SEとしての個人事業を継続しつつ、自社アプリ開発を目的としたマイクロ法人（合同会社）を設立するための実務ガイドである。
「行政のポンコツな対応」および「銀行の厳格審査」を想定し、**物理的接触をゼロ**にしながら、最長期間でスケジュールを算出したものである。

## 2. 設立・運用開始スケジュール（ワーストケース算出）

すべての工程をオンラインまたは郵送で完結させ、窓口での「水際作戦」を封殺する前提のフロー。

| 工程 | 手順・アクション内容 | 完了までの最長期間 | 累計期間（最長） | 備考・リスク対策 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 設立準備** | バーチャルオフィス契約、印鑑発注、個人の印鑑証明取得、資本金払込 | **約8日間** | **約8日** | 郵便遅延、印鑑作成の混雑を想定。 |
| **2. 登記申請** | freee等で書類作成し、法務局へ郵送またはオンライン申請 | **約1日間** | **約9日** | 24時間申請可能（受付は平日）。 |
| **3. 法務局審査** | 法務局による登記審査。繁忙期や担当者の処理遅延を想定 | **約14日間** | **約23日** | 通常は1週間だが、最長2週間を計上。 |
| **4. 証明書取得** | オンラインで謄本・印鑑証明を請求し、郵送で受取 | **約4日間** | **約27日** | 郵送往復のタイムラグを考慮。 |
| **5. 各種届出** | 税務署・自治体への届出。★e-Tax等で即日完了 | **約1日間** | **約28日** | オンラインなら即日完了。 |
| **6. 社保反映** | e-Gov等で申請。マイナポータルへの資格情報反映まで | **約21日間** | **約49日** | **紙の保険証は廃止。** システム登録の最長値。 |
| **7. 銀行審査** | ネット銀行による厳格審査。追加資料の要求等を想定 | **約28日間** | **約56日** | **全体のボトルネック。** 最長4週間を計上。 |
| **8. 運用開始** | 役員報酬振込、freee口座同期設定、Apple登録変更 | **約1日間** | **約57日** | 口座開設完了と同時に稼働可能。 |

**【算出結論】**
* **最短期間**: 約1ヶ月（30日前後）
* **最長期間（ワーストケース）**: **約2ヶ月（57日前後）**

## 3. ポンコツ担当者・審査への対抗戦略

### 3.1 「紙の保険証」廃止への対応
現在は紙の健康保険証は発行されない。
* **ゴール設定**: マイナポータル上で「健康保険証利用登録」が「登録済み」になり、資格情報が反映された時点を健康保険の利用開始とする。
* **注意**: 年金事務所側が「資格確認書（紙）」を送付しようとする場合があるが、マイナ保険証の利用を前提とすれば、システム反映をもって完了とみなす。

### 3.2 年金事務所への「先制攻撃」
「役員報酬が低い」「バーチャルオフィスである」という理由で難癖をつけられるのを防ぐため、以下の書類を最初から能動的に添付する。
* **必須添付**: 個人事業主としての直近の「確定申告書（第一表・第二表）」および「青色申告決算書」のコピー（受信通知付）。
* **目的**: 「生計維持の主軸は個人事業側にあり、法人は副事業である」という事実を物理的に突きつけ、相手に「どうやって生活しているのか」と考えさせる隙を与えない。

### 3.3 銀行審査（口座開設）の突破
バーチャルオフィス＋無形商材（アプリ開発）は最も審査が厳しい。
* **提出資料**: アプリの仕様書、開発画面（スクリーンショット）、Apple Developer Programの登録画面、個人のエンジニア経歴書を「事業実態の証明」としてセットで提出する。
* **ポイント**: 「これから作る」ではなく「既に開発・実績がある」ことを強調する。

## 4. 窓口ガチャを回避する「強制突破」の心得

1. **非対面原則**: 窓口には絶対に行かない。e-Gov、e-Tax、郵送（特定記録）のみを使う。
2. **受理義務の主張**: もし電話等で「預かれない」とごねられた場合は、「行政手続法第7条に基づく受理義務」を主張し、口頭ではなく「文書による不受理理由書」を要求する。
3. **アナログの餌**: ポンコツ担当者がIT用語を理解できない場合は、あえてExcelで作った「アナログな業務日報」や「ハンコを押した議事録」など、彼らが安心する形式の資料を1枚添えてやる。
