# マイクロ法人設立にかかる初期準備金（改訂版）

マイクロ法人設立から、アプリ売上が安定して入金されるまでに必要な資金の明細です。
資金は「①設立実費（個人からの持ち出し）」と「②資本金（法人の事業資金）」に大別されます。

## 1. 設立にかかる法定実費・諸経費
設立登記のために必ず発生する実費です。発起人（個人）が立て替えて支払い、設立後に法人の「創立費」として経費計上します。

- **登録免許税**：60,000円（合同会社の最低額。法務局へ納付）
- **電子定款作成代行料**：約5,000円（印紙代4万円を回避するため行政書士等へ依頼）
- **法人実印・銀行印作成費用**：約5,000円〜10,000円
- **印鑑証明書・登記事項証明書取得費**：約2,000円
- **合計概算**：**約72,000円〜77,000円**

## 2. 資本金の推奨額
**推奨額：1,500,000円〜2,000,000円**

以下の理由から、初期の余裕を持たせた資本金設定を強く推奨します。
1. **口座開設審査の突破**：マネーロンダリング対策が厳しいネット銀行において、「バーチャルオフィス＋低資本金」による審査落ちを回避するため。
2. **債務超過の防止**：赤字期間を代表者からの「役員借入金」で補填し続けることによる、決算書の悪化（実質的な債務超過）を防ぐため。

### 【参考：設立直後から発生するキャッシュアウト】
アプリの売上入金が安定するまでの間、法人からは以下の資金が確実に流出します。

- **インフラ・システム維持費（年額一括払い想定）**：**計 164,732円**
  - バーチャルオフィス年額：48,000円
  - freee3ツール（会計・申告・人事労務）年額合計：116,732円
- **毎月の固定支出**：**月額 約130,000円**
  - 役員報酬：100,000円
  - 社会保険料等の法人負担分：約15,000円〜20,000円
  - ※売上入金まで半年かかった場合、固定支出だけで約78万円の資金が消化されます。

## 3. 総合計（設立に向けて確保すべき現金）

| 項目 | 金額の目安 | 備考 |
| :--- | :--- | :--- |
| **①設立実費（個人支出）** | **約75,000円** | 登録免許税、印鑑代等（実質的な出費） |
| **②資本金（法人口座へ）** | **1,500,000円〜2,000,000円** | 初期のシステム費・役員報酬・社保支払いの原資 |
| **合計準備金** | **約1,575,000円〜2,075,000円** | ※資本金分は「個人口座から法人口座への資金移動」であり、資産が消滅するわけではありません。 |
